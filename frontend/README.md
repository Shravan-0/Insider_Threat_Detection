# In_Threat Frontend

Frontend application for the In_Threat Insider Threat Detection System.

## Technology Stack

* React.js
* Material UI
* Axios
* Recharts

## Development

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm start
```

Build for production:

```bash
npm run build
```

## Backend API

The frontend communicates with the In_Threat backend through the `REACT_APP_API_URL` environment variable.

Production deployment is configured through Vercel.
